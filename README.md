# 📊 Data Science Lab
A curated collection of Data Science and Machine Learning experiments, assignments, and projects completed during the **2026–2027 academic year**.
This repository serves as a structured record of practical work, covering fundamental concepts, algorithms, data analysis techniques, and machine learning implementations. It is intended for learning, experimentation, and academic reference.

---

## 📌 Objectives
* Build a strong foundation in Data Science and Machine Learning.
* Implement and analyze core algorithms using Python.
* Explore data preprocessing, visualization, and feature engineering techniques.
* Develop, evaluate, and compare machine learning models.
* Maintain a well-organized archive of laboratory experiments and coursework.
